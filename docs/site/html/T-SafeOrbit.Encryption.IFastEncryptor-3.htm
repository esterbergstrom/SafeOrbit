IFastEncryptor<(Of <(<'TResult, TInput, TKey>)>)> Interface

Defines a fast and cryptographically strong encryptor.

Use this class when you prefer the performance over security gained by slower actions.

It's faster than ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)> but also cryptographically strong. It's not stronger than ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)> as it takes less time to decrypt the data and does not use salt.

For slower and cryptographically stronger operations prefer ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)>
