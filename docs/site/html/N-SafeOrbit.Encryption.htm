Classes

AesEncryptor

AesEncryptor encryption implementation is designed to be slow.

Use it if you prefer stronger security over performance. Prefer BlowfishEncryptor if you want a faster alternative.

An implementation of AES 256 using a slow IKeyDerivationFunction so as to frustrate brute-force attack or dictionary attack on the password or passphrase input value.

BlowfishCbc
Default blowfish implementation with CBC cipher mode.
BlowfishEcb
Default blowfish implementation with ECB cipher mode.
BlowfishEncryptor

Block cipher: 64-bit block Variable key length: 32 bits to 448 bits Much faster than DES and IDEA.

IFastEncryptor<(Of <(<'TResult, TInput, TKey>)>)>

Defines a fast and cryptographically strong encryptor.

Use this class when you prefer the performance over security gained by slower actions.

It's faster than ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)> but also cryptographically strong. It's not stronger than ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)> as it takes less time to decrypt the data and does not use salt.

For slower and cryptographically stronger operations prefer ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)>
ISafeEncryptor

Defines the default ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)> using Byte arrays as the types.

ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)>

Defines a slow and cryptographically stronger encryptor.

Use this class when you prefer the security over performance.

It's stronger than IFastEncryptor<(Of <(<'TResult, TInput, TKey>)>)> because it takes more to decrypt and it uses an additional TSalt.

For faster and cryptographically strong operations prefer IFastEncryptor<(Of <(<'TResult, TInput, TKey>)>)>
