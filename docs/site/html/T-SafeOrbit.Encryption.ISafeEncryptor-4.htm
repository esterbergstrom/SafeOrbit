ISafeEncryptor<(Of <(<'TResult, TInput, TKey, TSalt>)>)> Interface

Defines a slow and cryptographically stronger encryptor.

Use this class when you prefer the security over performance.

It's stronger than IFastEncryptor<(Of <(<'TResult, TInput, TKey>)>)> because it takes more to decrypt and it uses an additional TSalt.

For faster and cryptographically strong operations prefer IFastEncryptor<(Of <(<'TResult, TInput, TKey>)>)>
