SafeObject<(Of <(<'TObject>)>)>

SafeObject<(Of <(<'TObject>)>)> uses and tracks any outside modifications of the TObject.

Provides thread safe access to its inner object.

Important: Use only ApplyChanges(Action<(Of <<'(TObject>)>>)) to change the state of your object. It'll authorize your access and apply them securely.

Its security is based on IInjectionDetector and alerts if unknown changes to the object occurs based on current SafeObjectProtectionMode.
