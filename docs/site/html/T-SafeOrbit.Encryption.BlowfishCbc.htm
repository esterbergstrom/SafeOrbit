Remarks

CBC mode encrypts each block of data in succession so that any changes in the data will result in a completly different ciphertext. Also, an IV is used so that encrypting the same data with the same key will result in a different ciphertext. CBC mode is the most popular mode of operation. Read more : http://en.wikipedia.org/wiki/Block_cipher_modes_of_operation
