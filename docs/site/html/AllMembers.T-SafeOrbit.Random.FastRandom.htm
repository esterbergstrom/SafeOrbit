StaticInstance
Static instance of FastRandom A static RNG that is used to generate seed values when constructing new instances of FastRandom. This overcomes the problem whereby multiple FastRandom instances are instantiated within the same tick count and thus obtain the same seed, that approach can result in extreme biases occuring in some cases depending on how the RNG is used.
