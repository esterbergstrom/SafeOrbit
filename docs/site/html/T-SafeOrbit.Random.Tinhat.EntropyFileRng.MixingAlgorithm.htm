EntropyFileRng..::..MixingAlgorithm Enumeration

It is important to mix new seed entropy into the pool, using some sort of mixing algorithm that will neither reduce the entropy in the pool, nor allow a maliciously crafted new seed material to reduce the entropy in the pool. Many possible ways to do this could be used - we just happen to implement the ones listed here.
